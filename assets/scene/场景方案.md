# 街道场景设计方案

本文档汇集了6种典型的街道场景设计方案，涵盖绿色街道、步行优先、活力街道、林荫街道、公交优先和宁静街道。

---

## 场景一：绿色街道

### 场景概述
**英文名称**: Green Street
**定位**: 生态优先、绿化丰富的绿色街道
**设计重点**: 行人和自行车优先，强调生态绿化设计
**建议语**: “生态优先，绿意盎然，行人、自行车友好的绿色街道”

### 横断面设计参数
| 参数 | 数值 |
|------|------|
| 人行道宽度 | 4.0m |
| 自行车道宽度 | 2.5m |
| 机动车道数 | 2 |
| 总路幅宽度 | 14m |
| 人行横道宽度 | 3m |
| 设施带宽度 | 1.5m |

### 设计要点
- **设计规范**: pedestrian_priority_v1
- **目标定位**: 生态优先 (Ecological Greening Priority)
- **密度**: 0.3
- **demand_level**:
  - 行人: 高 (high)
  - 自行车: 高 (high)
  - 公交: 低 (low)
  - 机动车: 低 (low)

### 植物配置方案
| 类型 | 数量 |
|------|------|
| 高大乔木 | 10棵 |
| 中等乔木 | 6棵 |
| 灌木 | 8棵 |
| 花卉/地被 | 80㎡ |
