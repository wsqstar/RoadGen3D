## 🧠 CLIP 特征向量 (154 个)

每个 3D 资产都预先计算了 CLIP 文本-图像特征向量，用于：
- **语义匹配**: 根据自然语言描述匹配最合适的资产
- **风格一致性**: 确保场景内资产风格统一
- **快速检索**: 无需运行时计算，直接加载 `.pt` 文件

| 属性 | 值 |
|:---|:---|
| 模型 | OpenAI CLIP ViT-B/32 |
| 向量维度 | 512 维 |
| 单文件大小 | ~1.3 KB |
| 总大小 | 616 KB |
| 存储格式 | PyTorch `.pt` |

## 📦 Git 追踪状态

### ✅ 已提交到 Git 的文件
- `data/real/latents/*.pt` - CLIP 特征向量 (616 KB)
- `data/real/*.jsonl` - 资产清单索引 (50 KB)
- `knowledge/graphRAG/` - GraphRAG 配置和部分索引

### ❌ 本地存储 (不提交到 Git)
- `data/real/meshes/*.glb` - 3D 资产 (49 MB)
- `assets/` - 2D 纹理和图模板 (4.5 GB)
- `models/` - AI 模型权重 (3.3 GB)
- `knowledge/graphRAG/output/` - 知识库索引 (可重新生成)

### 📦 获取完整资产
```bash
# 克隆仓库 (含子模块)
git clone --recurse-submodules https://github.com/GIStudio/RoadGen3D.git

# 下载 CLIP 模型
huggingface-cli download openai/clip-vit-base-patch32 \
  --local-dir models/clip-vit-base-patch32

# 3D 资产和 2D 纹理需从内部渠道获取
```

```
原始资产创建
    ↓

    ↓

    ↓

    ↓

    ↓

    ↓

```
